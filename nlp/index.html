大模型源自 NLP:LLM 与多模态

「大模型」更多指的是大语言模型——Large Language Model;它正是从传统自然语言处理技术发展而来。

LLM 是核心

课程核心是后续大模型内容,而大模型多数语境下即大语言模型(LLM)。

多模态是延伸

如今的大模型也能处理图片、音频、视频——即所谓多模态;但最初处理的是自然语言。

NLP 是根

大模型由传统 NLP 技术演进而来,学好 NLP 对理解大模型内容必不可少。

Seq2Seq:序列到序列架构

Seq2Seq 即序列到序列:输入一句话、输出一句话,典型场景是翻译(中文进、英文出);它并非新网络,而是基于 RNN/LSTM/GRU 搭建的结构更新。

Attention 机制:增强 Seq2Seq

Attention 为增强 Seq2Seq 架构而生:解决固定上下文向量的信息压缩困难与缺乏动态感知;有了它,翻译等任务效果更好。掌握 Seq2Seq 与 Attention 之后,学 Transformer 会轻而易举。
